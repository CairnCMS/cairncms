# CairnCMS

## Open Content & Data Infrastructure for Self-Hosted Teams

Use CairnCMS to model data, manage content, expose APIs, and automate workflows from one workspace built on SQL tables you control.

CairnCMS is a GPLv3, self-hosted workspace for SQL-backed applications. It combines a no-code admin, granular permissions, REST and GraphQL APIs, file management, dashboards, and workflow automation in one system.

## Key Features

- **Open by design** — GPLv3 licensed, self-hosted, and built for teams that want to run their own infrastructure.
- **SQL-first** — works with standard SQL tables you control.
- **Multi-database support** — supports PostgreSQL, MySQL, SQLite, and MariaDB.
- **Generated APIs** — exposes REST and GraphQL APIs for your collections.
- **Permission-scoped workspace** — gives each role the right interface for managing content, data, files, dashboards, and workflows.
- **Automation built in** — create workflows from data events, webhooks, schedules, and manual triggers.
- **Extensible by design** — customize interfaces, layouts, modules, hooks, endpoints, operations, themes, and more.
