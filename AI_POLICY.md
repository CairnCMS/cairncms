# AI Usage Policy

If you use AI tools in your contribution:
- You must review and approve every line of generated code before submitting
- You must understand what the code does and why it works
- Plans and intent should come from you, not from the model
- You are fully responsible for correctness, security, and adherence to project standards
- You must disclose AI use in the PR template

We will close without review:
- Automated submissions from unsupervised agents or bots
- PRs that show no evidence of human judgment
- Contributions where the author cannot explain the code when asked

***

# Rationale

CairnCMS is a small project with limited maintainer bandwidth. Every PR that requires back-and-forth over AI-generated submissions is time taken away from actual development and project progress. These rules exist so that AI can be used transparently as a tool without overwhelming the people maintaining the project.
